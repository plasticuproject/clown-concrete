About this site

For maximum security this site and all it's application dependencies are updated every 60 seconds.


The calculators on this site all use the calculator-wrapper python library to perform calculations. calculator-wrapper is a locally built/managed interface for the concrete-calculator python library pulled from FakePyPi. Both of these libraries are open-source and download links are provided below.
