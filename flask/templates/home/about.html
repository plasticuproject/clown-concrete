About this site

For maximum security this site and all it's application dependencies are updated and rebuilt every 60 seconds.


The calculators on this site all use the calculator-wrapper python library to perform calculations. calculator-wrapper is locally built using poetry-1.5.1. It is an interface for the concrete-calculator python library pulled from FakePyPi on port 42425. Both of these libraries are open-source and download links are provided below.
